# Data Directory

Data files are not tracked in git due to their size. Download them using one of these methods:

## Option 1: Use the notebook (recommended)
```bash
jupyter notebook 00_Data_Download.ipynb
```

## Option 2: Manual download

**COVIDSenti** (90k tweets):
```bash
mkdir -p COVIDSenti
cd COVIDSenti
wget https://raw.githubusercontent.com/usmaann/COVIDSenti/master/data/Final_data.csv -O COVIDSenti.csv
```

**METS-CoV**:
```bash
git clone https://github.com/YLab-Open/METS-CoV.git
```

data/
